# GZLM
This contains R code for running various generalized linear regression models
This code was used to present at the SSWR conference in 2016 in Washington, DC in a workshop presented by Wendy Zeitlin, Charles Auerbach, John Orme, and Matthew Cuellar
We covered a variety of models including testing for goodness of fit
For more information or to receive conference materials, please contact wendy.zeitlin@gmail.com
